# 🐱🐶 Cat and Dog Image Classification using CNN and TensorFlow

A Deep Learning based Image Classification project developed using Convolutional Neural Networks (CNN) with TensorFlow and Keras to classify images as Cats or Dogs.

---

## 🚀 Features
- CNN based Image Classification
- TensorFlow & Keras implementation
- Data Augmentation
- Accuracy and Loss Visualization
- Cat/Dog Prediction
- Model Saving and Reusability

---

## 🛠️ Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib

---

## 📂 Dataset Structure
The Dataset Link:-https://www.kaggle.com/datasets/tongpython/cat-and-dog 
training_set/
│
├── cats/
└── dogs/

test_set/
│
├── cats/
└── dogs/

---

## 📊 Project Workflow
1. Load and preprocess dataset
2. Apply data augmentation
3. Build CNN model
4. Train and validate model
5. Evaluate performance
6. Save trained model
